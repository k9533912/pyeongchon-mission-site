# 평촌교회 선교사이트

평촌교회 선교사이트 정적 웹페이지입니다.

## 포함된 기능

- 선교자료
- 선교소식
- 선교활동
- 열방기도회 기도제목 저장
- 게시판 글 저장

## 게시 방법

이 사이트는 서버 설치 없이 `index.html`, `styles.css`, `script.js` 파일만 올리면 열립니다.

가장 쉬운 방법은 Netlify Drop입니다.

1. https://app.netlify.com/drop 에 접속합니다.
2. 이 폴더의 `pyeongchon-mission-site.zip` 파일을 업로드합니다.
3. 생성된 공개 주소로 접속합니다.

주의: 현재 게시판과 열방기도회 저장 내용은 방문자 각자의 브라우저에 저장됩니다.
여러 사람이 같은 내용을 함께 보려면 추후 데이터베이스 연결이 필요합니다.
